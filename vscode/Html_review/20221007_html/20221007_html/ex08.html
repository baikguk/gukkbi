* position : 위치를 이동시키는 속성 1. static : 기본값, 위치가 자신의 원래의 위치에 고정되어 있음 2. relative : static(원래)위치에서 상대적 이동 3. absolute : body의 위치에서 절대적 이동 - 스크롤시 위치고정 4. fixed : body의 위치에서 절대적 이동 - 스크롤시 따라다님 * 위치이동속성 1. x축이동 - left 혹은 right 중 한 개만 선택해서 수치 작성 2. y축이동 - top 혹은 bottom 중 한 개만 선택해서 수치 작성 * z-index : position값을 받은 요소들끼리 겹칠 때 깊이 조절 가능 숫자가 높을 수록 화면 앞쪽에 보임
